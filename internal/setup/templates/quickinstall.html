Quick Install (Docker)

Get started quickly with Docker! This setup uses sensible defaults for all services. You only need to configure your admin account and domain name.

What's included:
  • PostgreSQL database (Docker)
  • Redis cache (Docker)
  • IPFS storage (Docker)
  • Email service (Docker Mailpit)
  • All services managed automatically
