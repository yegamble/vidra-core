# Features Documentation

This directory will contain documentation for specific features and capabilities.

## Planned Documentation

- **Live Streaming** - RTMP ingestion, HLS delivery, VOD conversion
- **Notifications** - Real-time notification system with automatic triggers
- **Video Import** - Import from 1000+ platforms via yt-dlp
- **Plugin System** - Hook-based architecture with security verification
- **Video Redundancy** - Cross-instance replication and health monitoring
- **WebTorrent P2P** - Browser-compatible P2P delivery
- **Analytics** - Video analytics, retention curves, channel statistics
