# Validation Quick Reference Card

## Quick Commands

```bash
# Run all validations
make validate-all

# Quick validation (format + lint only)
make validate-quick

# Individual checks
make fmt-check    # Check formatting
make fmt          # Auto-fix formatting
make lint         # Run linter
make test-unit    # Run unit tests
make build        # Build verification
```

## Common Fixes

```bash
# Fix formatting
make fmt

# Fix linting (auto-fixes)
make lint

# Fix YAML (manual)
# Edit files reported by yamllint

# Fix failing tests
make test-unit  # Run to see failures
# Debug and fix code
```

## Exit Codes

- `0` = All validations passed
- `1` = One or more failures
